# MC-Sugarcane-Juice-Mod
Mod that adds a very simple crafting recipe to turn sugarcane into sugarcane juice, making more use for sugarcane farms and an easy new source of hunger replenishment!

## 25/06/26
Today is the start of the project, item texture has been created and file directory layout has been created for Java edition. Possible port to Bedrock version in the future.
